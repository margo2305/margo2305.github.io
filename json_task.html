Анализ доклада Павла Новикова

Обоснование выбора темы

JSON — стандарт индустрии, а эффективная работа с ним в C++ — это баланс между удобством и скоростью, что критически важно для производительных систем.

Развернутый конспект

01:44 — Валидация UTF-8: Нельзя просто верить строке. Докладчик объясняет, почему ручная проверка корректности байтовых последовательностей необходима для безопасности парсера.
13:45 — Глубокая вложенность: Стандартная рекурсия может привести к переполнению стека. Решение: использование стека в куче (heap) для обработки структур любой глубины.
17:23 — Оптимизация чисел: Традиционные методы (sprintf) медленные. Использование std::to_chars из C++17 позволяет записывать числа в буфер максимально быстро и без лишних аллокаций.
32:05 — Архитектура библиотеки: Обзор min_json_cpp. Как 1500 строк кода могут конкурировать с гигантами индустрии за счет узкой специализации.
