Итоговая работа: Сериализация в JSON

Описание: В данной работе представлена программная модель формирования данных (на примере чека книжного магазина) и их перевод в универсальный формат JSON.

Программа демонстрирует работу с библиотекой nlohmann/json.

// Исходный код C++
#include <iostream>
#include <string>
#include <nlohmann/json.hpp>

using json = nlohmann::json;

int main() {
    json book = {
        {"title", "C++ Programming"},
        {"year", 2026},
        {"is_available", true}
    };
    
    std::cout << book.dump(4) << std::endl;
    return 0;
}
