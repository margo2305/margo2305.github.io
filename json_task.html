Итоговая работа: Сериализация в JSON

Описание: В данной работе представлена программная модель формирования данных (на примере чека книжного магазина) и их перевод в универсальный формат JSON.

Программа демонстрирует корректную работу с библиотекой nlohmann/json и формирование иерархической структуры данных в C++.

Нажмите кнопку "Run" в интерактивном окне ниже, чтобы увидеть результат генерации JSON-кода в реальном времени:

// Исходный код: Сериализация объекта Book в JSON
                    #include <iostream>
                    #include <string>
                    #include <nlohmann/json.hpp>

                    using json = nlohmann::json;

                    int main() {
                        json book = {
                            {"title", "C++ Programming"},
                            {"year", 2026},
                            {"is_available", true}
                        };
                    
                        std::cout << book.dump(4) << std::endl;
                        return 0;
                    }
