JSON и сериализация в C++

Анализ доклада Павла Новикова

Обоснование выбора темы

Выбор данной темы продиктован тем, что JSON сегодня — это стандарт де-факто для обмена данными в веб-сервисах и распределенных системах. В мире C++ эффективная работа с текстом всегда была вызовом. Доклад Павла Новикова интересен тем, что он рассматривает не просто использование готовых решений, а глубокую оптимизацию «под капотом»: как минимизировать аллокации памяти, использовать современные инструкции процессора и добиваться производительности, превосходящей стандартные парсеры в разы. Для меня, как для студента ИТ-направления, это отличный пример того, как знание архитектуры C++ превращается в конкретные миллисекунды выигрыша скорости.

Код: JSON in C++

#include <iostream>
#include "min_json.hpp"

int main() {
    // Создаем объект JSON
    min_json::object book = {
        {"title", "JSON in C++"},
        {"pages", 1500},
        {"author", "P. Novikov"}
    };
    
    std::cout << "Serialized: " << book.dump() << std::endl;
    return 0;
}
